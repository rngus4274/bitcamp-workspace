##### 저장소 및 프로젝트 폴더 준비하기

1. 저장소로 사용할 폴더 생성
   c:\User\bitcamp\bitcamp-java-project

2. 폴더를 git저장소로 만든다.

3. 깃헙에 bitcamp-java-project 저장소를 생선한다.

4. 로컬 저장소를 원격 저장소에 연결

5. gradle을 이용하여 저장소를 프로젝트 폴더로 만든다.

6. .gitignore 파일 준비하기

- bitcamp-workspace에서 사용하는 파일을 그대로 복사한다.

7. 로컬 저장소에 백업 한 후 원격 저장소에 업로드
